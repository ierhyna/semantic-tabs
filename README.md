# Semantic Tabs with Pure JavaScript

[Demo](http://jsfiddle.net/ierhyna/c9hwgdyz/show)

The HTML structure is the following:

`
<div class="tabs">
    <div class="tab">
        <h3 class="tab-header">Tab 1</h3>
        <div class="tab-text"></div>
    </div>
    ...
    <div class="tab">
        <h3 class="tab-header">Tab 5</h3>
        <div class="tab-text"></div>
    </div>
</div>
`
